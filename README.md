Скрипт ищет все коммиты в репозитории, где в сообщении встречается заданное слово (по умолчанию — `правк`).
Полезен, если нужно быстро найти все фиксы, “правки” и прочие потуги коллег по истории репозитория.

Скрипт берёт настройки из переменных окружения: 
- OWNER - владелец репозитория (как организация, так и юзер) 
- REPO - имя репозитория
- BRANCH - целевая ветка
- TOKEN - GitHub-токен для авторизации
